--Projet de reconnaissance des émotions faciales avec le CNN
Face-Emotions — Reconnaissance d’émotions faciales (Flask + Keras + MediaPipe)

Système complet de détection et classification d’émotions à partir d’images ou de flux webcam, basé sur :

Détection de visage temps réel avec MediaPipe,

Prétraitement OpenCV,

CNN personnalisé (Keras/TensorFlow) entraîné sur FER2013,

Application Flask (upload d’image + mode webcam).
